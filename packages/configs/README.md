# @sytizen/configs

Configuration files for sytizen packages.

## Installation

```sh
npm add -D @sytizen/configs
```

## Usage

Add the following in your `biome.json`:

```json
{
	"$schema": "./node_modules/@biomejs/biome/configuration_schema.json",
	"extends": ["@sytizen/configs/biome.json"]
}
```
