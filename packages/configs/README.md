# @sytizen/configs

Opinionated configuration files for `@sytizen/*` packages

## Usage

### Biome

In your `biome.json(c)`:

```json
{
	"extends": ["@sytizen/configs/biome.json"]
}
```

### TypeScript

in your `tsconfig.json`:

```json
{
	"extends": ["@sytizen/configs/tsconfig.json"]
}
```
